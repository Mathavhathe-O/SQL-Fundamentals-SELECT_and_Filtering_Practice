# SQL-Fundamentals-SELECT_-_Filtering_Practice
This repository contains my first hands-on SQL exercise focused on retrieving and filtering data using core SQL commands. The project is based on a structured dataset and demonstrates how to write queries and predict outputs — a key skill for any data analyst.

## 📌 Project Overview

The goal of this exercise was to build a strong foundation in SQL by working with a single dataset (`employees`) and solving 10 practical query problems.

Each question required:
- Writing a correct SQL query using proper syntax
- Applying filtering logic using WHERE conditions
- Predicting and drawing the expected output table

## 🧠 Key Skills Demonstrated

- Writing SQL queries using SELECT  
- Filtering data using WHERE, AND, OR, NOT, and IN  
- Sorting results with ORDER BY (ASC & DESC)  
- Removing duplicates using DISTINCT  
- Limiting results using LIMIT  
- Understanding how query structure affects output  

## 🗂 Dataset Overview

The dataset used is a simple `employees` table with the following columns:

- id  
- first_name  
- last_name  
- department  
- salary  
- hire_date  
- city
